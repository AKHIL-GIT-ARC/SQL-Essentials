# Introduction to SQL

## What is SQL?

SQL (Structured Query Language) is a standardized programming language used to communicate with relational databases. It allows users to create, retrieve, update, delete, and manage data stored in tables.

SQL is supported by almost every major Relational Database Management System (RDBMS), including:
- MySQL
- PostgreSQL
- SQLite
- Microsoft SQL Server
- Oracle Database

---

## Why Learn SQL?

SQL is one of the most important skills for:

- Software Development
- Backend Development
- Data Analysis
- Data Science
- Machine Learning
- Artificial Intelligence
- Business Intelligence
- Database Administration
Almost every application stores its data inside a database, making SQL an essential skill.

---

## What Can SQL Do?

SQL can:

- Create databases
- Create tables
- Insert records
- Retrieve data
- Update existing records
- Delete records
- Filter data
- Sort data
- Combine multiple tables
- Perform calculations
- Manage database security

---

## SQL Categories

### DDL (Data Definition Language)

Used to define database structures.
Commands:
- CREATE
- ALTER
- DROP
- TRUNCATE
- RENAME

---

### DML (Data Manipulation Language)

Used to modify data.
Commands:
- INSERT
- UPDATE
- DELETE

---

### DQL (Data Query Language)

Used to retrieve data.
Command:
- SELECT

---

### DCL (Data Control Language)

Used for permissions.
Commands:
- GRANT
- REVOKE

---

### TCL (Transaction Control Language)

Used to control transactions.
Commands:
- COMMIT
- ROLLBACK
- SAVEPOINT

---

## SQL is Not Case Sensitive

These are equivalent:

```sql
SELECT * FROM students;
```

```sql
select * from students;
```

However, using uppercase for SQL keywords is considered best practice.

## Best Practices

- Write SQL keywords in uppercase.
- Use meaningful table names.
- Keep queries readable.
- Use indentation for complex queries.
- Comment important queries.

---

## Common Mistakes

- Forgetting semicolons.
- Misspelling table names.
- Confusing SQL with programming languages.
- Assuming SQL is case-sensitive.

---

## Summary

- SQL stands for Structured Query Language.
